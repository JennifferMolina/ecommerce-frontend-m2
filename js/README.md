# TecnoStore - Tienda Online

Este proyecto corresponde al desarrollo de un MVP (Producto Mínimo Viable) de una tienda en línea.

El sitio fue desarrollado utilizando HTML5 semántico, Bootstrap para el diseño responsivo y JavaScript básico para la interacción del carrito de compras.

## Funcionalidades

- Página de inicio con productos en formato cards
- Página de detalle de producto
- Carrito de compras simulado
- Contador de productos en el carrito
- Diseño responsivo para móviles y escritorio

## Tecnologías utilizadas

- HTML5
- Bootstrap 5
- JavaScript
- Git y GitHub
